# Model_detection_maladies-_tomate
Le projet "Model_detection_maladies_tomate" vise à développer un modèle de vision par ordinateur capable de détecter et de classifier les maladies affectant les feuilles de tomate.
 Ce projet s'inscrit dans le cadre de l'agriculture de précision, où l'utilisation de technologies avancées permet d'améliorer la santé des cultures et d'optimiser les rendements.
